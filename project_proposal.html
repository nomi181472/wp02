Project Proposal

Description

Our project is based on Exercise and Nutrition Facts MANAGEMENT SYSTEM. In this project, we provide many services to the user. The user have to register to the system by providing necessary information. The information will be stored in secure database. Whenever the user want to access system, he must login with their registered information. The home page contains necessary information about the system and the services that the system provides.

Functionality

  1. Authorization
  2. Set Custom Exercise
  3. Set Custom Diet
  4. Tracking Diet and Exercise
  5. Reminder
  6. Progress Snapshot

Development Tasks

  1. Gathering Requirements.
  2. Adding Functionality to the interface by adding database and API.
